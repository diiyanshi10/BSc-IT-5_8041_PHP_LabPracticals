QUICK START GUIDE
================================================================================

1. DATABASE SETUP
   - Open DEPLOYMENT_GUIDE.md for complete step-by-step instructions
   - Create MySQL database on InfinityFree
   - Import database.sql file using phpMyAdmin
   - Note down database credentials

2. FILE UPLOAD
   - Download all files from this folder
   - Connect via FTP to your InfinityFree account
   - Upload to public_html directory
   - Maintain folder structure (admin/, api/, assets/)

3. CONFIGURATION
   - Edit config.php with your database credentials:
     * DB_HOST: sql.infinityfree.com
     * DB_USER: your_db_username
     * DB_PASS: your_db_password
     * DB_NAME: your_db_name
     * SITE_URL: https://yourdomain.com/
   - Save and upload updated config.php

4. CREATE ADMIN ACCOUNT
   - Register at: https://yourdomain.com/register.php
   - Go to phpMyAdmin > users table
   - Change 'role' from 'customer' to 'admin'
   - Login as admin to access dashboard

5. ACCESS YOUR STORE
   - Frontend: https://yourdomain.com/
   - Admin: https://yourdomain.com/admin/
   - Add products and start selling!

================================================================================
                        SUPPORT & HELP
================================================================================

1. DEPLOYMENT ISSUES?
   → Read DEPLOYMENT_GUIDE.md (complete step-by-step guide)
   → Check InfinityFree Control Panel for database info
   → Verify FTP credentials in connection settings

2. DATABASE CONNECTION ERROR?
   → Verify credentials in config.php
   → Check database was imported successfully
   → Ensure database user has proper permissions

3. PAGE NOT FOUND?
   → Verify all files uploaded to public_html
   → Check folder structure matches original
   → Ensure .htaccess is uploaded

4. IMAGES NOT SHOWING?
   → Add product images via admin panel
   → Verify image URLs in database
   → Check assets/images/ folder exists

5. NEED MORE HELP?
   → InfinityFree Support: https://www.infinityfree.net/support/
   → PHP Docs: https://www.php.net/manual/
   → MySQL Docs: https://dev.mysql.com/doc/

================================================================================
                        SECURITY NOTES
================================================================================

⚠ IMPORTANT: Before going live:

1. Change all default passwords
2. Update credentials in config.php with strong passwords
3. Enable HTTPS in InfinityFree settings
4. Update SITE_URL to use https://
5. Keep backups of your database
6. Never commit credentials to version control
7. Regularly monitor for suspicious activity
